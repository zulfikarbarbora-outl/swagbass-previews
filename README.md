# Swagbass — Preview MP3s

Swag bass electric bass samples

## About this repo

This repo holds **128 kbps MP3 previews** of every sample in the lossless library
[`zulfikarbarbora-outl/karoryfer.swagbass`](https://github.com/zulfikarbarbora-outl/karoryfer.swagbass).

Previews are used by the web DAW for fast streaming / low-bandwidth playback.
For export / lossless playback the DAW fetches the original WAV/FLAC from the
lossless repo.

## Stats

- **Files**: 341 MP3 previews
- **Bitrate**: 128 kbps CBR
- **Sample rate**: 44.1 kHz (down-mixed to mono/stereo by ffmpeg)
- **Source branch**: `master`
- **License**: CC0-1.0 (same as source — see source repo for full text)
- **Author**: Karoryfer Samples

## URLs

Every MP3 is reachable as a raw GitHub URL (Fastly CDN-backed):

```
https://raw.githubusercontent.com/zulfikarbarbora-outl/swagbass-previews/master/<filename>.mp3
```

The full file catalog (with parsed note / velocity / round-robin info) is in
[`index.json`](./index.json).

## Verification

Run [`verify.py`](./verify.py) to check that every MP3 in `index.json` returns HTTP 200:

```bash
python3 verify.py
```

## Regeneration

These previews were generated by
[`tools/freesound/preview-pipeline.py`](https://github.com/zulfikarbarbora-outl/web-daw-samples/blob/main/tools/freesound/preview-pipeline.py)
in the `web-daw-samples` repo. Re-running that script will skip files that
already exist and only download/convert new ones.
